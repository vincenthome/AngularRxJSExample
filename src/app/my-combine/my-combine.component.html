Combine Items Streams -> Array Single Stream

combineLatest
  • Function:
  • 1. When all Input Streams emitted at least once
  • 2. Returns LATEST emitted of ALL Input Streams into an Observable Array
  • Use case: Re-evaluate state when an Action occurs
forkJoin
  • Function:
  • 1. When all Input Streams Completed
  • 2. Returns the LAST emitted from ALL COMPLETE Input Streams into an Observable Array and Completes
  • Use case: Wait until ALL http request Input Streams Complete and combine all results
withLatestFrom
  • Operator:
  • 1. When all Input streams emitted at least once
  • 2. Only when Source stream emitted
  • 3. Returns LATEST emitted of ALL Input Streams into an Observable Array
  • 4. Output Stream Completes when ALL Input Streams Complete
  • Use case: Use 1 Input Stream to 'Regulate' the Output Stream
